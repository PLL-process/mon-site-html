💻 Cours complet HTML / CSS / JavaScript

Niveau : 4e

Démarche pédagogique : ☑ Démarche d’investigation

📘 Présentation de l’activité

Comment créer une page Web lisible, bien structurée, moderne, et pédagogique ? Ce cours progressif permet aux élèves de 4e et 3e de comprendre les bases du HTML, CSS et JavaScript tout en manipulant des outils modernes (CodePen, GitHub Pages, etc.)

🎯 Compétences mobilisées

  • Comprendre la structure d’une page web (HTML)
  • Appliquer un style via CSS (couleurs, texte, tableaux...)
  • Différencier class et id
  • Utiliser des sélecteurs CSS, des styles, des codes hexadécimaux
  • Créer des interactions simples avec JavaScript

📚 Ressources à lire avant de coder

  • MDN – HTML
  • MDN – CSS
  • W3Schools – Tutoriels HTML/CSS/JS

📌 Partie 1 – HTML

<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="UTF-8">
    <title>Ma page</title>
  </head>
  <body>
    <h1>Bienvenue</h1>
    <p>Ceci est ma première page HTML</p>
  </body>
</html>

🎨 Partie 2 – CSS

body {
  background-color: #121212;
  color: white;
  font-family: Verdana, sans-serif;
}
.titre {
  color: red;
  text-align: center;
}

🧪 Partie 3 – Activité de codage sur CodePen ou GitHub Pages

Crée une page HTML avec un titre, un paragraphe, une liste à puces, un lien vers un site externe, une image, et applique un style CSS personnalisé.
